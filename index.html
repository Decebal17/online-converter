100% on-device · No uploads

How it works & why it’s private

Your files are converted directly in your browser. There’s no server upload: data stays on your device and is never sent to us.

Tech in plain words

  • On-device processing: browser APIs and WebAssembly handle conversion locally.
  • No network transfer: file bytes aren’t sent to any server for conversion.
  • Temporary memory only: data lives in RAM while the tab is open; close/refresh to clear.

Privacy guarantees

  • No server copy of your files, ever.
  • No account required and minimal metadata.
  • Security headers reduce leakage (Referrer/Permissions policies).
